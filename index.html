工作流

AI 不越权，用户有最后控制权
1. 解析意图

用户用一句话描述收款、兑换、授权管理等目标，AI 输出结构化草稿。

2. 风险拦截

检查地址、金额、网络、授权范围、钓鱼关键字和异常手续费。

3. 清晰确认

把最终交易内容翻译成用户能理解的语言，而不是只展示哈希和十六进制。

4. 本地签名

真实产品中由 Token Core 类本地钱包核心管理 Keystore 与签名，不把私钥交给 AI。
